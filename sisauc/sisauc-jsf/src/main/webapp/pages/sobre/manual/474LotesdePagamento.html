4.7.4 Lotes de Pagamento

4 Menus do Sistema ›› 4.7 Auditoria Retrospectiva ››

Gera uma lista de itens de GAB auditados como conforme, para posteriormente gerar um lote de pagamento para um determinado credenciado.


Ao clicar em Lotes de Pagamento é exibida a tela 'Gerenciar Lotes', que apresenta uma lista com os lotes já gerados.

Tela 'Gerenciar Lotes'

Caso o usuário queira imprimir o relatório do lote ou informar a Nota Fiscal deve-se clicar no botão Visualizar, é apresentada então a tela Gerenciar Lote.

Fig.39 Tela 'Gerenciar Lote'

Caso o usuário queira alterar a nota fiscal basta clicar no botao () onde será aberta a janela abaixo para que sejam editados os dados desejados, disponibilizando as opções de () e ():

Fig.39 Tela 'Gerenciar Lote'

Caso o usuário clique em () o sistema volta para a Tela de "Gerenciar Lote".
