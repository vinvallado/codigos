4.1.1 Usuários

4 Menus do Sistema ›› 4.1 Cadastros ››

USUÁRIOS são os operadores do Sistema.


Clicando em 'Usuários' surgirá a tela 'Cadastro de Usuários' (Fig.3) onde será apresentada uma lista dos usuários cadastrados.


É possível facilitar a busca de usuários utilizando os filtros OM, Login e Nome Completo, bastando apenas clicar e digitar a palavra a ser buscada, ou selecionar as opções Perfis e Ativo.

Fig.3 Tela 'Cadastro de Usuários'

O botão() exibe o(s) perfi(l)(s) associado(s) a um usuário.

Janela que exibe os perfis vinculados ao usuário

O botão () permite desativar e o botão () ativar um usuário.

Ao desativar ou ativar um usuário uma mensagem de confirmação será exibida.

Editar Usuário

O botão () permite editar um usuário.

Tela para edição do usuário.

Para ativar ou desativar o usuário clique no checkbox ativo.

Não é possível editar o login de um usuário previamente cadastrado.

A senha pode ser 'resetada' através do botão (). A senha ficará igual ao login.

Cadastrar novo usuário

O botão () permitirá a inserção de novo usuário na tela 'Formulário de Usuário' (Fig.4).

Fig.4 Tela 'Formulário de Usuário'

Preencher os campos conforme o que se segue:


Nome completo: Com iniciais maiúsculas e sem abreviaturas.

Login: Deve ser formado pelo 'nome de guerra' seguido de todas as iniciais.


Exemplo:


Nome completo: Alberto Santos Dumont

Login: dumontasd


E-mail: Poderá ser o do COMAER ou outro desejado pelo usuário.

Organização Militar: Selecionar a OM do usuário.

Perfil: Os perfis associados ao usuário podem ser cumulativos. A associação dos perfis será estabelecida conforme as ações a serem desempenhadas por cada usuário de acordo com a tabela 1.


Após a definição do Perfil, clicar em (). A senha inicial de cada usuário será igual ao login e deverá ser alterada no primeiro acesso.
