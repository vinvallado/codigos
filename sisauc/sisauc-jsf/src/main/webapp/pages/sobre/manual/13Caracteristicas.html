1.3 Características

1 Dados do Sistema ››

O SISAUC foi desenvolvido na linguagem de programação JAVA, tendo como gerenciador de banco de dados o PostgreSQL. É um sistema web e multiplataforma com armazenamento de dados local, sendo, portanto, uma solução rápida e eficiente.


O Sistema é sincronizado à Base de Dados do SIGPES, através de um aplicativo de Web Service, para obter as informações dos beneficiários e de seus dependentes.


O Sistema informa os valores devidos à coparticipação dos beneficiários do FUNSA ao AGC Aerconsig, também por outro aplicativo de Web Service.
