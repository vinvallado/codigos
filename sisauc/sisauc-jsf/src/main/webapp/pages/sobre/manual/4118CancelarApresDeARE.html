4.11.3 Cancelar Apres. de GAB

4 Menus do Sistema ›› 4.11 DIRSA ››

Gera uma lista ítens de GAB auditados, fins de serem cancelados conforme a necessidade.


Ao clicar no ítem de menu à esquerda Cancelar Apres. de GAB é exibida a tela 'Cancelamento de Apresentação de GAB' que apresenta uma lista com os ítens de GAB com um dos seguintes estados: 'Em auditoria' e 'Apresentada' disponíveis para o cancelamento de uma apresentação, conforme imagem abaixo:



Caso o usuário queira cancelar uma apresentação GAB este deve clicar no botão e, em seguida, confirmar o cancelamento.
